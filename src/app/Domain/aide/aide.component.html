Comment faire un signalement ?

Cliquer sur le bouton S dans le menu ensuite choissisez le type d'incident que vous souhaitez signaler. Renseignez l'heure et le lieu de l'incident. Faites une description de l'incident si vous le souhaitez. Confirmez votre signalement et voila vous avez aider les autres Spotters. Vous pouvez aussi ajouter cette fonctionnalités en widget pour plus de facilité.
